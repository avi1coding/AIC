# **App Name**: AI Homework Checker

## Core Features:

- Content Upload: Allow users to upload homework content via direct text input, PDF upload, or image upload.
- AI-Powered Analysis: Analyze the uploaded content using AI to detect if it was AI generated.
- Results Display: Clearly display the results of the AI analysis, highlighting potential AI-generated content.
- Confidence Score: Provide a confidence score indicating the likelihood that the homework was AI-generated; this involves a tool that determines how the confidence score is derived.
- Content selection: Let the user select which homework analysis method to use (image, direct text, PDF, etc).

## Style Guidelines:

- Primary color: Dark indigo (#4B0082) for a sophisticated and intelligent feel.
- Background color: Light gray (#F0F0F0) for a clean and professional look.
- Accent color: Teal (#008080) to highlight important analysis results and calls to action.
- Body text: 'PT Sans', a sans-serif for readability and a modern touch.
- Headline text: 'Space Grotesk' a sans-serif for prominent titles. 
- Use clean, simple icons to represent different content types and analysis results.
- Maintain a clean, intuitive layout with clear sections for content input, analysis, and results.
- Use subtle animations to provide feedback during the analysis process and when displaying results.
